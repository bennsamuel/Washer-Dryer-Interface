# Summary

A very simple box for Arduino PIR sensor. Now you can attach your sensors easily on every drop ceiling.

Also I've included a simple mount for wall (using a screw to fix it). Inside the box you can pin up the sensor using some 'blu tack'. You won't need screw or any additional holes to put your PIR inside.

Editable file in Sketchup is included.

Thank you for collecting, like and commenting.

Any suggestions are so welcome!
